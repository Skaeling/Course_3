<h1> Код для виджета «Операции по счетам»</h1>

<h3>Описание</h3>
Функция выводит список из 5 последних выполненных клиентом банковских операций в формате:

"дата перевода" "описание перевода"

"откуда" -> "куда"

"сумма перевода" "валюта"
